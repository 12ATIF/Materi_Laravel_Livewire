Tahap 4: Kalkulasi & Logika Payroll

🚀 Tutorial Proyek ⏱️ 50 menit 🧮 Praktik Lifecycle Hook

📌 Prasyarat

  • Konsep Lifecycle Hooks Livewire (updated()).
  • Berhasil menyelesaikan Tahap 3 sehingga tabel Karyawan di *database* sudah terbentuk.

🎯 Tujuan Belajar

Materi tahap 4 ini adalah Core / Inti dari nama aplikasi kita. Kamu akan mampu:

  • Menggunakan Lifecycle Hook updated() untuk menciptakan kalkulator perhitungan otomatis layaknya Microsoft Excel (Gaji Pokok + Tunjangan - Potongan).
  • Membuat skema database untuk merekam riwayat penggajian karyawan per bulan.

📂 1. Tabel Database Penggajian (Payroll)

Kita memerlukan satu tabel lagi untuk merekam struk gaji yang nantinya akan dibayarkan setiap bulan kepada karyawan.

Terminal 
# Membuat Model Payroll beserta migrasinya
php artisan make:model Payroll -m

Buka file *_create_payrolls_table.php di folder `database/migrations` dan tambahkan struktur berikut:

*_create_payrolls_table.php 
public function up(): void
{
    Schema::create('payrolls', function (Blueprint $table) {
        $table->id();
        // Menyambungkan tabel ini dengan tabel employees (Foreign Key)
        $table->foreignId('employee_id')->constrained()->cascadeOnDelete();
        
        // Komponen angka - Decimal 15 digit dengan 2 angka di belakang koma untuk presisi Uang
        $table->decimal('basic_salary', 15, 2)->default(0);   // Gaji Pokok
        $table->decimal('allowance', 15, 2)->default(0);      // Total Tunjangan
        $table->decimal('deduction', 15, 2)->default(0);      // Total Potongan
        $table->decimal('net_salary', 15, 2)->default(0);     // Gaji Bersih Terakhir (Take Home Pay)
        
        $table->string('month_year');                         // Contoh: "April 2026"
        $table->timestamps();
    });
}

Buka app/Models/Payroll.php dan pasang $fillable serta Relasi Balik ke Employee:

app/Models/Payroll.php 
class Payroll extends Model
{
    protected $fillable = ['employee_id', 'basic_salary', 'allowance', 'deduction', 'net_salary', 'month_year'];

    // Setiap slip gaji ini adalah kepunyaan SATU karyawan.
    public function employee()
    {
        return $this->belongsTo(Employee::class);
    }
}

Jalankan php artisan migrate untuk memproses perintah ini ke tabel MySQL Anda.

🧮 2. Sihir Kalkulator Pada Livewire Component

Di Tahap 1 kita sudah mendesain static/payroll.blade.php di mana terdapat form Gaji Pokok, Tunjangan, dan Potongan. Tugas kita sekarang adalah membuat otak di baliknya sehingga Gaji Bersih otomatis terhitung saat angkanya diketik.

Terminal 
# Membuat komponen transaksi Gaji
php artisan make:livewire PayrollCalculator

Penggunaan Hook updated() untuk Agregasi:

Buka app/Livewire/PayrollCalculator.php:

app/Livewire/PayrollCalculator.php 
namespace App\Livewire;

use Livewire\Component;
use App\Models\Employee;
use App\Models\Payroll;

class PayrollCalculator extends Component
{
    // Deklarasi input variable
    public $employee_id;
    public $basic_salary = 0;
    public $allowance = 0;
    public $deduction = 0;
    
    // Output Variable (read-only secara logika)
    public $net_salary = 0;

    // =======================================================================
    // LIFECYCLE HOOKS SAKTI DARI LIVEWIRE
    // =======================================================================
    // Fungsi di bawah ini dipanggil OTOMATIS detik demi detik 
    // setiap kali user mengubah nilai di input HTML!
    public function updated($propertyName)
    {
        // Lakukan Kalkulasi: THP = (Gaji + Tunjangan) - Potongan
        // Cast ke (int) untuk memastikan matematika berjalan lancar meski valuenya string kosong.
        $this->net_salary = ((int)$this->basic_salary + (int)$this->allowance) - (int)$this->deduction;
    }

    // Proses Menyimpan Data Struk ke Database MySQL
    public function savePayroll()
    {
        $this->validate([
            'employee_id' => 'required|exists:employees,id',
            'basic_salary' => 'required|numeric|min:0',
        ]);

        Payroll::create([
            'employee_id' => $this->employee_id,
            'basic_salary' => $this->basic_salary,
            'allowance' => $this->allowance,
            'deduction' => $this->deduction,
            'net_salary' => $this->net_salary,
            'month_year' => date('F Y'), // Ambil bulan & tahun hari ini
        ]);

        // Tampilkan alert sukses dan kosongkan kalkulator
        session()->flash('success', 'Berhasil menerbitkan Slip Gaji!');
        $this->reset(['employee_id', 'basic_salary', 'allowance', 'deduction', 'net_salary']);
    }
