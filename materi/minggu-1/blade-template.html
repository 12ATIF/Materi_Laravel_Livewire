1.4 Blade Template

📗 Minggu 1 ⏱️ 25 menit

🎯 Tujuan Belajar

  • Memahami Blade sebagai template engine Laravel
  • Menampilkan variabel dengan {{ }}
  • Menggunakan @if, @foreach, dan @extends

Apa itu Blade?

Blade adalah template engine bawaan Laravel. Dengan Blade, kamu bisa:

  • Menampilkan data PHP dalam HTML dengan mudah
  • Menggunakan kondisi dan perulangan
  • Membuat layout yang reusable

Menampilkan Variabel

welcome.blade.php 
<!-- Tampilkan variabel -->
<h1>Halo, {{ $nama }}</h1>

<!-- Tampilkan dengan default value -->
<p>{{ $email ?? 'Tidak ada email' }}</p>

Kondisi @if

profile.blade.php 
@if($isAdmin)
    <span>Kamu adalah Admin</span>
@else
    <span>Kamu adalah User biasa</span>
@endif

Perulangan @foreach

books.blade.php 
<ul>
@foreach($books as $book)
    <li>{{ $book->title }} - {{ $book->author }}</li>
@endforeach
</ul>

Layout dengan @extends

layouts/app.blade.php 
<html>
<head>
    <title>@yield('title')</title>
</head>
<body>
    @yield('content')
</body>
</html>
home.blade.php 
@extends('layouts.app')

@section('title', 'Halaman Home')

@section('content')
    <h1>Selamat Datang!</h1>
@endsection
💡

Tips

Gunakan layout untuk bagian yang sama di semua halaman (navbar, footer). Hemat kode!

📝 Latihan

  1. Buat view baru siswa.blade.php
  2. Tampilkan daftar nama siswa dengan @foreach
  3. Tambahkan kondisi: jika tidak ada siswa, tampilkan "Data kosong"
