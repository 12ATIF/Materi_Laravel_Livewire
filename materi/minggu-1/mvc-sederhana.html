Apa itu MVC?

MVC adalah cara mengorganisir kode supaya rapi. Seperti divisi kerja di restoran:

📊

Model

Gudang bahan & resep
= Database & Logic

👁️

View

Penyajian makanan
= Tampilan HTML

🎮

Controller

Pelayan yang koordinasi
= Penghubung

Alur MVC di Laravel

Alur Request 
User klik tombol
    ↓
Route → terima request, kirim ke Controller
    ↓
Controller → ambil data dari Model
    ↓
Model → query ke database
    ↓
View → tampilkan ke user

Contoh Kode

routes/web.php 
Route::get('/buku', [BookController::class, 'index']);
BookController.php 
public function index()
{
    $books = Book::all(); // Model
    return view('books.index', ['books' => $books]); // View
}
💡

Ingat!

Model = Data, View = Tampilan, Controller = Penghubung. Jangan campur-campur!

  1. Gambarkan diagram alur MVC dengan kata-katamu sendiri
  2. Sebutkan file apa saja yang termasuk Model, View, dan Controller di Laravel
