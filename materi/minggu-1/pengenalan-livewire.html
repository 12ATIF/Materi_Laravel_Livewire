1.5 Pengenalan Livewire

📗 Minggu 1 ⏱️ 20 menit

🎯 Tujuan Belajar

  • Memahami apa itu Livewire dan fungsinya
  • Mengetahui cara kerja reaktivitas Livewire
  • Membandingkan Livewire dengan JavaScript framework

Apa itu Livewire?

Livewire adalah full-stack framework untuk Laravel yang memungkinkan kamu membuat UI interaktif tanpa menulis JavaScript.

💡

Analogi Sederhana

Bayangkan kamu punya asisten yang bisa memahami dan menjalankan perintahmu tanpa kamu harus berbicara dalam bahasa asing (JavaScript). Kamu cukup berbicara PHP!

Bagaimana Cara Kerjanya?

Alur Kerja Livewire 
1. User klik tombol di browser
      ↓
2. Livewire kirim request AJAX ke server
      ↓
3. PHP method dijalankan di server
      ↓
4. Livewire kirim balik HTML baru
      ↓
5. Browser update hanya bagian yang berubah

Livewire vs JavaScript Framework

Aspek Livewire React/Vue
Bahasa PHP saja ✅ JavaScript
Learning Curve Mudah ✅ Lebih kompleks
Integrasi Laravel Sempurna ✅ Perlu setup API
SEO Bagus ✅ Perlu SSR

Contoh Sederhana

Counter.php 
class Counter extends Component
{
    public $count = 0;
    
    public function tambah()
    {
        $this->count++;
    }
}

Hanya dengan kode PHP di atas, kamu sudah bisa membuat counter yang interaktif tanpa JavaScript sama sekali! 🎉
