1.3 Instalasi Laravel

📗 Minggu 1 ⏱️ 30 menit 🔧 Praktik

🎯 Tujuan Belajar

  • Menginstall Composer dan Laragon
  • Membuat project Laravel baru
  • Menjalankan development server

1️⃣ Install Laragon

Laragon adalah paket lengkap untuk development PHP di Windows. Download di laragon.org

📌

Kenapa Laragon?

Sudah include PHP, MySQL, Apache, dan Composer. Satu kali install, langsung siap pakai!

2️⃣ Buat Project Laravel

Buka Terminal di Laragon (klik kanan > Terminal), lalu ketik:

Terminal 
# Masuk ke folder www
cd C:\laragon\www

# Buat project baru
composer create-project laravel/laravel perpustakaan

# Masuk ke folder project
cd perpustakaan

# Jalankan server
php artisan serve

3️⃣ Akses di Browser

Buka browser dan ketik: http://localhost:8000

Kalau muncul halaman "Laravel" berarti sukses! 🎉

4️⃣ Install Livewire

Terminal 
# Install Livewire
composer require livewire/livewire

# Cek versi
php artisan livewire:version
⚠️

Troubleshooting

Kalau error "composer not found", pastikan Laragon sudah running dan buka terminal lewat Laragon.

📝 Latihan

  1. Install Laragon di komputer masing-masing
  2. Buat project Laravel baru dengan nama "belajar-livewire"
  3. Install Livewire dan screenshot hasilnya
