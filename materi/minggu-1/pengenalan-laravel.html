1.1 Pengenalan Laravel

📗 Minggu 1 ⏱️ 20 menit 📖 Teori

📌 Prasyarat

Sebelum memulai materi ini, pastikan kamu sudah menguasai:

  • HTML & CSS dasar — bisa membuat halaman web sederhana
  • PHP dasar — paham variabel, fungsi, array, dan form handling
  • Database dasar — pernah pakai phpMyAdmin atau MySQL sederhana

💡 Belum menguasai? Tidak apa-apa! Materi ini akan menjelaskan dari dasar. Tapi akan lebih mudah jika sudah punya fondasi di atas.

🎯 Tujuan Belajar

Setelah menyelesaikan materi ini, kamu akan mampu:

  • Menjelaskan apa itu framework dan mengapa developer profesional menggunakannya
  • Menyebutkan minimal 5 keunggulan Laravel dibanding PHP murni
  • Membedakan kode PHP murni vs kode Laravel untuk tugas yang sama
  • Memahami ekosistem Laravel (Artisan, Composer, Blade, Eloquent)

💡 Apa itu Framework?

Sebelum kita bicara Laravel, kamu perlu paham dulu: apa itu framework?

🏗️

Analogi: Bangun Rumah

Bayangkan kamu mau bikin rumah. Ada dua cara:

  1. Cara manual — Buat batu bata sendiri, aduk semen, potong kayu, desain sendiri. Bisa, tapi sangat lama dan rawan salah.
  2. Pakai kontraktor + material jadi — Sudah ada kerangka rumah, tinggal pilih desain, cat, dan isi furnitur.

Framework = pilihan kedua. Semua "kerangka dasar" sudah disediakan, kamu tinggal fokus bikin fitur aplikasi.

Dalam pemrograman web, framework adalah kumpulan kode yang sudah jadi dan terstruktur, yang menyediakan:

📁

Struktur Folder

Kode terorganisir rapi, tidak berserakan di satu folder

🔧

Tool Bawaan

Fitur siap pakai: login, database, email, dll

🛡️

Keamanan

Perlindungan dari serangan SQL injection, XSS, CSRF

🚀 Kenapa Laravel?

Ada banyak framework PHP di dunia: CodeIgniter, Symfony, Yii, CakePHP. Tapi Laravel adalah yang paling populer. Kenapa?

✅ 1. Sintaks Elegan & Mudah Dibaca

Laravel dibuat agar kodenya mirip bahasa Inggris. User::find(1) artinya "cari user dengan ID 1". Sangat intuitif!

✅ 2. Fitur Bawaan Super Lengkap

Login/register, kirim email, upload file, antrian job, API — semua sudah tersedia tanpa install tambahan!

✅ 3. Komunitas Terbesar di Dunia

Jutaan developer, ribuan package gratis, dan tutorial di mana-mana. Kalau stuck, pasti ada jawabannya.

✅ 4. Artisan — Asisten Pribadimu

Laravel punya command-line tool bernama Artisan. Cukup ketik perintah, dan file-file otomatis dibuat. Sangat menghemat waktu!

✅ 5. Dipakai Perusahaan Besar

Banyak perusahaan besar dan startup menggunakan Laravel: dari e-commerce sampai fintech. Belajar Laravel = skill yang dicari di dunia kerja!

⚔️ Perbandingan: PHP Murni vs Laravel

Berikut perbandingan nyata — tugas yang sama, cara yang berbeda:

Contoh 1: Ambil Data dari Database

❌ Dengan PHP Murni (4 baris, rawan error):

php-murni.php 
// Koneksi manual ke database
$conn = mysqli_connect("localhost", "root", "", "db_sekolah");

// Query manual (rawan SQL Injection!)
$sql = "SELECT * FROM users WHERE id = " . $_GET['id'];
$result = mysqli_query($conn, $sql);
$user = mysqli_fetch_assoc($result);

✅ Dengan Laravel (1 baris, aman & bersih):

laravel.php 
// Satu baris, otomatis aman dari SQL Injection!
$user = User::find($id);
🔍

Penjelasan

PHP Murni: Kamu harus tulis koneksi manual, query SQL mentah, dan ambil data manual. Bahkan, kode di atas rawan SQL Injection karena langsung menyambung variabel ke query!

Laravel: Cukup panggil User::find($id). Laravel secara otomatis membuat query yang aman dengan prepared statements. Tidak perlu tulis koneksi — itu sudah diatur di file konfigurasi.

Contoh 2: Validasi Form

Aspek PHP Murni Laravel
Cek field kosong if(empty($_POST['nama'])) 'nama' => 'required'
Cek format email filter_var($email, FILTER_VALIDATE_EMAIL) 'email' => 'required|email'
Tampilkan error Manual: buat array error, kirim balik Otomatis: @error('nama')
Jumlah kode ~30+ baris ~5 baris

🧩 Ekosistem Laravel

Laravel bukan hanya satu tool — ia punya ekosistem lengkap yang saling mendukung. Berikut yang akan kamu pelajari:

⚡

Artisan

Command line tool untuk membuat file, migration, menjalankan server, dll.

📦

Composer

Package manager PHP. Seperti "toko alat" untuk install library tambahan.

🎨

Blade

Template engine untuk membuat tampilan HTML dengan sintaks PHP yang ringkas.

🗄️

Eloquent ORM

Cara elegan berinteraksi dengan database tanpa menulis SQL manual.

🔌

Livewire

Membuat halaman interaktif tanpa JavaScript! Ini yang akan kita pelajari.

🛣️

Routing

Mengatur URL mana yang mengarah ke halaman mana. Simpel dan fleksibel.

🗺️

Peta Perjalanan Belajarmu

Jangan khawatir dengan banyaknya istilah! Kamu akan mempelajarinya satu per satu, bertahap. Di Minggu 1, kita fokus di dasar. Minggu 2, kita bikin fitur interaktif. Minggu 3, kita gabungkan semua jadi project nyata!

🏢 Siapa yang Pakai Laravel?

Laravel bukan framework mainan — ini digunakan oleh perusahaan dan platform besar:

🏬

E-Commerce

Banyak toko online dibangun dengan Laravel

🏥

Sistem Informasi

Rumah sakit, sekolah, dan instansi pemerintah

💼

Startup

Startup Indonesia banyak yang pakai Laravel

⚠️ Kesalahan Umum Pemula

❌ "Saya harus mahir PHP dulu sebelum belajar Laravel"

Tidak perlu mahir 100%. Cukup paham dasar PHP (variabel, fungsi, array, if-else). Laravel justru akan mempercepat pemahamanmu tentang PHP karena kamu langsung praktik proyek nyata.

✅ Solusi: Mulai saja! Kamu akan belajar PHP lebih dalam SAMBIL belajar Laravel.

❌ "Laravel terlalu sulit, lebih baik pakai PHP murni saja"

Memang awalnya terasa asing. Tapi setelah terbiasa, Laravel justru jauh lebih mudah dan cepat. Seperti belajar naik sepeda — awalnya jatuh terus, tapi setelah bisa, lebih cepat dari jalan kaki!

✅ Solusi: Ikuti materi ini bertahap. Dalam 3 minggu kamu sudah bisa bikin aplikasi CRUD!

❌ "Framework akan membuat saya tidak paham cara kerjanya"

Justru sebaliknya. Framework yang baik (seperti Laravel) mengajarkan pola-pola pemrograman yang benar: MVC, ORM, routing, dll. Ini skill yang berlaku di framework manapun!

✅ Solusi: Sambil pakai Laravel, coba pahami "di balik layar" apa yang terjadi. Kita akan bahas ini di materi berikutnya.

📋 Ringkasan Materi

  • Framework = kerangka kerja yang menyediakan struktur dan tool siap pakai
  • Laravel = framework PHP paling populer di dunia, sintaks elegan dan fitur lengkap
  • Keunggulan Laravel: kode rapi, aman, cepat development, komunitas besar
  • Ekosistem Laravel: Artisan, Composer, Blade, Eloquent, Routing, Livewire
  • Livewire = extension Laravel yang akan kita pelajari untuk buat web interaktif tanpa JavaScript
