1.1 Pengenalan Laravel

📗 Minggu 1 ⏱️ 15 menit 📖 Teori

🎯 Tujuan Belajar

  • Memahami apa itu framework dan kenapa perlu menggunakannya
  • Mengenal Laravel sebagai framework PHP modern
  • Memahami keunggulan Laravel dibanding PHP murni

Apa itu Framework?

Bayangkan kamu mau bikin rumah. Ada dua pilihan:

  1. Bikin dari nol - Cetak batu bata sendiri, bikin semen, potong kayu
  2. Pakai material jadi - Beli batu bata, semen, kayu yang sudah siap pakai

Framework itu seperti pilihan kedua. Kumpulan kode yang sudah jadi dan siap pakai, jadi kamu tinggal fokus bikin fitur aplikasimu.

💡

Analogi Sederhana

Framework = Kerangka rumah yang sudah jadi. Kamu tinggal dekorasi dan isi furnitur sesuai kebutuhan.

Kenapa Pakai Laravel?

Laravel adalah framework PHP paling populer di dunia. Ini alasannya:

✅ Kode Lebih Rapi

Struktur folder terorganisir. Ada tempat khusus untuk database, tampilan, dan logika.

✅ Fitur Bawaan Lengkap

Authentication, database, email, file storage - semua sudah disediakan.

✅ Komunitas Besar

Banyak tutorial, package, dan bantuan kalau stuck.

Perbandingan: PHP Murni vs Laravel

Query database dengan PHP murni:

php-murni.php 
$conn = mysqli_connect("localhost", "root", "", "db");
$sql = "SELECT * FROM users WHERE id = " . $id;
$result = mysqli_query($conn, $sql);
$user = mysqli_fetch_assoc($result);

Query yang sama dengan Laravel:

laravel.php 
$user = User::find($id);
📌

Perhatikan!

1 baris vs 4 baris. Laravel punya ORM (Eloquent) yang bikin query database jadi sangat singkat dan aman.
