1.6 Struktur Component Livewire

📗 Minggu 1 ⏱️ 25 menit 🔧 Praktik

🎯 Tujuan Belajar

  • Memahami anatomi component Livewire
  • Membuat component Livewire pertama
  • Menghubungkan class PHP dengan Blade view

Anatomi Component

Setiap component Livewire terdiri dari 2 file:

📄 Class PHP

app/Livewire/NamaComponent.php

Berisi logic, data, dan method

🎨 Blade View

resources/views/livewire/nama-component.blade.php

Berisi tampilan HTML

Membuat Component

Terminal 
php artisan make:livewire Counter

Perintah di atas akan membuat 2 file:

  • app/Livewire/Counter.php
  • resources/views/livewire/counter.blade.php

Contoh Component

app/Livewire/Counter.php 
namespace App\Livewire;

use Livewire\Component;

class Counter extends Component
{
    public $count = 0;
    
    public function increment()
    {
        $this->count++;
    }
    
    public function render()
    {
        return view('livewire.counter');
    }
}
livewire/counter.blade.php 
<div>
    <h1>{{ $count }}</h1>
    <button wire:click="increment">Tambah</button>
</div>

Menggunakan Component

welcome.blade.php 
<!-- Cara 1: Tag component -->
<livewire:counter />

<!-- Cara 2: Blade directive -->
@livewire('counter')
⚠️

Penting!

Jangan lupa tambahkan @livewireStyles dan @livewireScripts di layout utama!

📝 Latihan

  1. Buat component HelloWorld yang menampilkan "Hello, World!"
  2. Tambahkan property nama dan tampilkan "Hello, [nama]!"
  3. Buat tombol untuk mengubah nama menjadi nama sendiri
