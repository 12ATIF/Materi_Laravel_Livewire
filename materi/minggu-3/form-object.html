Catatan Guru RPL â€” Prinsip Clean Code

Di dunia kerja, kode yang "berhasil jalan" saja tidak cukup. Kode harus mudah dibaca, dipahami, dan dimaintain oleh orang lain. Kalau satu class punya 100+ baris dan banyak tanggung jawab, itu tanda harus dipecah. Ini disebut prinsip Single Responsibility Principle (SRP).

Perhatikan Prefix "form."!

Saat menggunakan Form Object, semua wire:model dan @error harus pakai prefix form.:

  • Sebelum: wire:model="judul" â†’ Sesudah: wire:model="form.judul"
  • Sebelum: @error('judul') â†’ Sesudah: @error('form.judul')

âœï¸ Form Object untuk Edit (fill)

Salah satu keunggulan Form Object adalah method fill() untuk mengisi data dari model Eloquent saat mode edit:

app/Livewire/Forms/BukuForm.php â€” Tambahan method 
class BukuForm extends Form
{
    public ?Buku $buku;  // Model yang sedang diedit

    // ... property dan validate seperti sebelumnya ...

    // Method untuk set data dari model ke form
    public function setBuku(Buku $buku)
    {
        $this->buku = $buku;

        // fill() mengisi semua property dari model
        $this->fill(
            $buku->only(['judul', 'penulis', 'isbn', 'tahun', 'kategori', 'stok', 'deskripsi'])
        );
    }

    // Method untuk update data
    public function update()
    {
        $this->validate();

        $this->buku->update(
            $this->all()
        );
    }

    // Method untuk create data
    public function store()
    {
        $this->validate();

        return Buku::create(
            $this->all()
        );
    }
}
app/Livewire/ManajemenBuku.php â€” Create & Edit 
class ManajemenBuku extends Component
{
    public BukuForm $form;
    public $showForm = false;
    public $isEditing = false;

    // Buka form untuk tambah buku baru
    public function create()
    {
        $this->form->reset();  // Kosongkan form
        $this->isEditing = false;
        $this->showForm = true;
    }

    // Buka form untuk edit buku
    public function edit(Buku $buku)
    {
        $this->form->setBuku($buku);  // Isi form dari model
        $this->isEditing = true;
        $this->showForm = true;
    }

    // Simpan (create atau update)
    public function simpan()
    {
        if ($this->isEditing) {
            $this->form->update();
            session()->flash('message', 'Buku berhasil diupdate!');
        } else {
            $this->form->store();
            session()->flash('message', 'Buku berhasil ditambahkan!');
        }

        $this->showForm = false;
        $this->form->reset();
    }
}

Catatan Guru RPL â€” Kapan Pakai Form Object?

  • Form sederhana (1-3 field) â†’ tidak perlu Form Object, biasa saja cukup
  • Form menengah (4-7 field) â†’ direkomendasikan pakai Form Object
  • Form kompleks (8+ field) â†’ WAJIB pakai Form Object
  • Form yang sama dipakai di banyak tempat â†’ WAJIB pakai Form Object
