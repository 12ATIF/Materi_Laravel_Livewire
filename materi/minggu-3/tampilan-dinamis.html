3.4 Tampilan Dinamis

📕 Minggu 3 ⏱️ 25 menit 🔧 Teori + Praktik

📌 Prasyarat

  • Sudah paham wire:click dan wire:model
  • Sudah paham Blade Template (@if, @foreach)

🎯 Tujuan Belajar

  • Menampilkan/menyembunyikan elemen dengan Blade conditional
  • Membuat tab navigation dinamis
  • Mengubah class CSS secara dinamis
  • Menggunakan wire:loading untuk indikator proses
  • Memahami wire:transition untuk animasi

🤔 Apa Itu Tampilan Dinamis?

🎭

Analogi: Panggung Teater

  • 🎭 Tampilan dinamis = panggung teater → properti (latar panggung, aktor) berubah tiap adegan
  • 🎬 wire:click = sutradara → memberi perintah "ganti adegan!"
  • 💡 Property PHP = naskah → menentukan adegan mana yang ditampilkan
  • 🔄 Re-render = perpindahan adegan → halaman berubah tanpa buka-tutup tirai (tanpa refresh)

1️⃣ Show/Hide dengan @if

Cara paling dasar: tampilkan atau sembunyikan elemen berdasarkan property:

app/Livewire/ToggleDemo.php 
class ToggleDemo extends Component
{
    public $showDetail = false;

    public function toggle()
    {
        $this->showDetail = !$this->showDetail;  // Toggle true ↔ false
    }
}
View 
<button wire:click="toggle">
    {{ $showDetail ? '🔽 Sembunyikan' : '▶️ Lihat Detail' }}
</button>

@if($showDetail)
    <div style="padding:1rem;background:#F3F4F6;border-radius:8px;margin-top:0.5rem">
        <p>Ini konten detail yang muncul/menghilang! 🪄</p>
    </div>
@endif

2️⃣ Tab Navigation Dinamis

app/Livewire/TabView.php 
class TabView extends Component
{
    public $activeTab = 'profil';  // Tab aktif saat ini
}
View — Tab dengan class dinamis 
<div>
    <!-- Tombol tab -->
    <div style="display:flex;gap:0.5rem;margin-bottom:1rem">
        @foreach(['profil', 'nilai', 'absensi'] as $tab)
            <button
                wire:click="$set('activeTab', '{{ $tab }}')"
                style="padding:0.5rem 1rem;border-radius:8px;border:none;cursor:pointer;
                       {{ $activeTab === $tab
                           ? 'background:#3B82F6;color:white'
                           : 'background:#E5E7EB;color:#374151' }}">
                {{ ucfirst($tab) }}
            </button>
        @endforeach
    </div>

    <!-- Konten tab -->
    <div style="padding:1.5rem;background:#F9FAFB;border-radius:8px">
        @if($activeTab === 'profil')
            <h3>👤 Data Profil Siswa</h3>
            <p>Nama, kelas, alamat...</p>
        @elseif($activeTab === 'nilai')
            <h3>📊 Tabel Nilai</h3>
            <p>Nilai UTS, UAS, tugas...</p>
        @elseif($activeTab === 'absensi')
            <h3>📅 Rekap Absensi</h3>
            <p>Hadir, izin, sakit, alpha...</p>
        @endif
    </div>
</div>
💡

Trik: $set tanpa bikin method

wire:click="$set('activeTab', 'nilai')" → langsung ubah property tanpa perlu bikin method PHP khusus!

3️⃣ wire:loading — Indikator Proses

Tampilkan loading saat Livewire sedang memproses request:

Berbagai cara pakai wire:loading 
<!-- ① Tampilkan saat SEMUA proses -->
<div wire:loading>⏳ Loading...</div>

<!-- ② Tampilkan saat method TERTENTU -->
<div wire:loading wire:target="save">💾 Menyimpan data...</div>

<!-- ③ Sembunyikan saat loading -->
<button>
    <span wire:loading.remove>💾 Simpan</span>
    <span wire:loading>⏳ Menyimpan...</span>
</button>

<!-- ④ Ubah class saat loading -->
<button wire:loading.class="opacity-50">Simpan</button>

<!-- ⑤ Disable button saat loading -->
<button wire:loading.attr="disabled">Simpan</button>
Directive Fungsi
wire:loading Tampilkan elemen saat loading
wire:loading.remove Sembunyikan elemen saat loading
wire:loading.class="..." Tambah class CSS saat loading
wire:loading.attr="disabled" Tambah HTML attribute saat loading
wire:target="save" Hanya aktif saat method tertentu diproses

4️⃣ Dynamic CSS Class

Contoh: Status badge dinamis 
<!-- Warna badge berubah berdasarkan status -->
@foreach($books as $book)
    <span style="padding:0.25rem 0.75rem;border-radius:999px;font-size:0.8rem;
        {{ $book->status === 'tersedia'
            ? 'background:#D1FAE5;color:#065F46'
            : 'background:#FEE2E2;color:#991B1B' }}">
        {{ $book->status }}
    </span>
@endforeach

5️⃣ Contoh Real-World: Dashboard Siswa

Mari gabungkan semua konsep dalam satu komponen lengkap:

app/Livewire/StudentDashboard.php 
class StudentDashboard extends Component
{
    public $activeTab = 'overview';
    public $showNotif = true;

    public function dismissNotif()
    {
        $this->showNotif = false;
    }
}
View — Dashboard kombinasi semua teknik

⏱️ wire:poll — Auto Refresh

Ingin data yang otomatis ter-update tanpa user harus klik? Gunakan wire:poll! Komponen akan otomatis refresh setiap beberapa detik.

Auto-refresh setiap 5 detik 
<!-- Komponen ini otomatis refresh setiap 5 detik -->
<div wire:poll.5s>
    <p>🕐 Terakhir update: {{ now()->format('H:i:s') }}</p>
    <p>📚 Jumlah buku: {{ $totalBuku }}</p>
</div>

<!-- Bisa juga panggil method spesifik -->
<div wire:poll.10s="refreshData">
    <!-- Memanggil method refreshData() setiap 10 detik -->
    <p>Notifikasi baru: {{ $unreadCount }}</p>
</div>

<!-- Hanya polling saat tab aktif (visible) -->
<div wire:poll.5s.visible>
    <!-- Tidak memakan resource saat tab di background -->
</div>
📢

Catatan Guru RPL — Jangan Berlebihan!

wire:poll mengirim request ke server terus-menerus. Gunakan hanya jika benar-benar perlu (dashboard monitoring, notifikasi). Tambahkan .visible agar hanya aktif saat tab terlihat.

🧭 wire:navigate — Navigasi SPA

Secara default, pindah halaman di Laravel = full page reload. Dengan wire:navigate, perpindahan halaman terasa seperti Single Page Application (SPA) — kilat tanpa loading putih!

wire:navigate 
<!-- Link biasa: full page reload (lambat) -->
<a href="/dashboard">Dashboard</a>

<!-- Dengan wire:navigate: SPA-like (kilat!) -->
<a href="/dashboard" wire:navigate>Dashboard</a>

<!-- Prefetch saat hover — lebih cepat lagi! -->
<a href="/books" wire:navigate.hover>📚 Daftar Buku</a>
🚀

Bagaimana wire:navigate Bekerja?

  • Livewire mengambil halaman baru di background (AJAX)
  • Hanya mengganti isi <body>, tidak reload seluruh halaman
  • CSS & JS yang sudah dimuat tidak perlu diunduh ulang
  • Terasa instan — seperti React/Vue tapi tanpa build step!

Tambahkan @persist di layout untuk menyimpan elemen yang tidak berubah (sidebar, player musik, dll).

🎨 wire:transition — Animasi Halus

Tambahkan animasi saat elemen muncul/menghilang menggunakan wire:transition:

Contoh wire:transition 
<!-- Elemen ini muncul/hilang dengan animasi fade -->
@if($showNotification)
    <div wire:transition
         style="padding:1rem;background:#D1FAE5;border-radius:8px;">
        ✅ Berhasil disimpan!
    </div>
@endif

<!-- Hanya animasi saat masuk (enter) -->
@if($showDetail)
    <div wire:transition.in>Detail buku...</div>
@endif

⚠️ Kesalahan Umum Pemula

❌ Lupa bahwa @if menghapus elemen, bukan menyembunyikan

@if(false) → elemen dihapus total dari HTML. Jika butuh tetap ada tapi hidden, gunakan CSS.

✅ Pakai style="display:{{ $show ? 'block' : 'none' }}" jika perlu.

❌ Tidak pakai wire:target pada loading

Loading muncul untuk semua aksi, bukan cuma yang dimaksud.

✅ Tambahkan wire:target="methodName" untuk spesifik.

❌ Typo nama tab/property

$activeTab === 'profill' (double L) → tab tidak pernah aktif.

✅ Gunakan constant atau array agar tidak salah ketik.

📋 Ringkasan

  • @if/$showDetail → Show/Hide toggle
  • $activeTab + @if/@elseif → Tab navigation dinamis
  • wire:loading + wire:target → Indikator proses spesifik
  • wire:loading.class / .attr → Ubah tampilan saat loading
  • $set('prop', 'value') → Ubah property tanpa bikin method
