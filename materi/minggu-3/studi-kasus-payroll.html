3.7 THE MASTERCLASS: Aplikasi Payroll (Full Code + Export PDF)

📕 Minggu 3 ⏱️ 200 menit 👨‍💻 Mega Project SMK Terarah
⚙️ Versi: 🔥 Laravel 11.x ⚡ Livewire 3.x 🐘 PHP 8.2+ 📄 dompdf / barryvdh-laravel-dompdf Project ini integrasikan package eksternal (dompdf untuk export PDF). Versi package terbaru biasanya kompatibel dengan Laravel 11.

Alur Lengkap Pembuatan Website! Banyak tutorial di Internet yang membingungkan karena kodenya separuh-separuh. Di halaman keramat ini, kita membongkar seluruh File Project asli (Sesuai Standar Software House), mulai dari Install Plugin, Migrasi Database, Layouting Induk, 4 Komponen Utama, Fitur Download Cetak PDF, sampai Rute Aksesnya! Ikuti secara kronologis.

1 Persiapan Plugin & Fitur PDF

Sebelum ngoding, pastikan Laravel kita dilengkapi senjata tambahan. Disini aplikasi Payroll kita mewajibkan fitur Export PDF Slip Gaji Karyawan.

Jalankan terminal dan ketik perintah ini untuk mendownload DOMPDF ciptaan Barryvdh:
composer require barryvdh/laravel-dompdf
Setelah itu instal Livewire:
composer require livewire/livewire

2 Database & Migrations

Aturan pertama ngoding: Atur tempat tidurnya dulu (Database). Jalankan php artisan make:model Employee -m dan php artisan make:model Payroll -m

A. Tabel Karyawan ..._create_employees_table.php

Schema::create('employees', function (Blueprint $table) {
    $table->id();
    $table->string('name');           // Nama karyawan
    $table->string('position');       // Jabatan
    $table->decimal('base_salary', 15, 2); // Gaji pokok
    $table->timestamps();
});

B. Tabel Payroll ..._create_payrolls_table.php

Schema::create('payrolls', function (Blueprint $table) {
    $table->id();
    $table->foreignId('employee_id')->constrained()->cascadeOnDelete();
    $table->decimal('allowance', 15, 2)->default(0);    // Pinjaman/Tunjangan
    $table->decimal('deduction', 15, 2)->default(0);    // Potongan
    $table->decimal('total_salary', 15, 2); 
    $table->date('pay_date');                            // Tgl pencairan
    $table->string('status')->default('unpaid');         // paid atau unpaid
    $table->timestamps();
});
Jika file di atas berhasil kamu buat, langsung tancap gas perintah php artisan migrate.

3 Logika Relasi Database (Models)

Laravel butuh file model agar tahu cara ngobrol satu sama lain antar tabel.

A. Model Employee app/Models/Employee.php

<?php
namespace App\Models;
use Illuminate\Database\Eloquent\Model;

class Employee extends Model {
    protected $fillable = ['name', 'position', 'base_salary'];

    // Info Relasi: "Karyawan ini punya banyak (HasMany) data slip gaji loh"
    public function payrolls() {
        return $this->hasMany(Payroll::class);
    }
}

B. Model Payroll app/Models/Payroll.php

<?php
namespace App\Models;
use Illuminate\Database\Eloquent\Model;

class Payroll extends Model {
    protected $fillable = ['employee_id', 'allowance', 'deduction', 'total_salary', 'pay_date', 'status'];
    protected $casts = ['pay_date' => 'date'];

    // Info Relasi: "Slip gaji ini miliknya (BelongsTo) Siapa?"
    public function employee() {
        return $this->belongsTo(Employee::class);
    }
}

🟡 Modul B: Employee Manager (Daftar Karyawan)

Logika: Bisa Tambah, Ngabari Pesan Sukses, Validasi Form, dan Live Cari!

class EmployeeManager extends Component {
    // Variable public wajib
    public $name = '', $position = '', $base_salary = '', $search = '';
    public $editingId = null, $showForm = false, $successMessage = '';

    // Validator laravel biasa
    protected function rules() {
        return [
            'name' => 'required', 'position' => 'required', 'base_salary' => 'required|numeric'
        ];
    }

    public function create() { $this->showForm = true; $this->editingId = null; }

    public function save() {
        $this->validate(); // Validasi mati!
        if ($this->editingId) {
            Employee::find($this->editingId)->update(['name'=>$this->name, 'position'=>$this->position, 'base_salary'=>$this->base_salary]);
            $this->successMessage = 'Diperbarui!';
        } else {
            Employee::create(['name'=>$this->name, 'position'=>$this->position, 'base_salary'=>$this->base_salary]);
            $this->successMessage = 'Dibuat!';
        }
        $this->showForm = false; // Tutup menu form
    }

    public function edit($id) {
        $e = Employee::find($id);
        $this->editingId = $e->id; $this->name = $e->name; 
        $this->position = $e->position; $this->base_salary = $e->base_salary;
        $this->showForm = true;
    }

    public function delete($id) { Employee::find($id)->delete(); }

    public function render() {
        $db = Employee::latest();
        // Wajib di-assign ulang! where() mengembalikan instance baru, tidak mengubah $db
        if($this->search) $db = $db->where('name', 'like', "%{$this->search}%");

        return view('livewire.employee-manager', ['employees' => $db->get()])->layout('layouts.app');
    }
}

🟡 Modul C: Payroll Calculator (Hitung Otomatis)

Backend Logika (PayrollCalculator.php). Disini menarik: Setiap kali kita ketik besaran potongan, totalnya bisa menghitung real-time.

class PayrollCalculator extends Component {
    public $employee_id = '', $allowance = '0', $deduction = '0', $status = 'unpaid';
    public $baseSalary = 0, $totalSalary = 0;

    // Ajaib! Saat Select Box berubah, Fungsi updatedEmployeeId otomatis terpanggil
    public function updatedEmployeeId($value) {
        $e = Employee::find($value);
        $this->baseSalary = $e ? $e->base_salary : 0;
        $this->calculateTotal();
    }

    public function updatedAllowance() { $this->calculateTotal(); }
    public function updatedDeduction() { $this->calculateTotal(); }

    private function calculateTotal() {
        $this->totalSalary = $this->baseSalary + (int)$this->allowance - (int)$this->deduction;
    }

    public function save() {
        Payroll::create([
            'employee_id' => $this->employee_id,
            'allowance' => $this->allowance, 'deduction' => $this->deduction,
            'total_salary' => $this->totalSalary, 'status' => $this->status,
            'pay_date' => now()
        ]);
        $this->reset();
    }

    public function render() { return view('livewire.payroll-calculator', ['employees' => Employee::all()])->layout('layouts.app'); }
}

🟡 Modul D: Payroll History (Menampilkan Data + Tombol Status)

Logika: Flip Status Paid / Unpaid Hanya dengan menekan tombol (toggle).

class PayrollHistory extends Component {
    public $filterStatus = '';

    public function toggleStatus($id) {
        $p = Payroll::find($id);
        $p->status = ($p->status == 'paid') ? 'unpaid' : 'paid';
        $p->save();
    }

    public function render() {
        $db = Payroll::with('employee')->latest();
        // Wajib di-assign ulang! where() mengembalikan instance baru, tidak mengubah $db
        if ($this->filterStatus) $db = $db->where('status', $this->filterStatus);

        return view('livewire.payroll-history', ['payrolls' => $db->get()])->layout('layouts.app');
    }
}

6 Fitur Spesial: Export Slip PDF

Bagian inilah mengapa kita tadi menginstal DOMPDF. Berbeda dari Livewire, untuk membuat file cetak PDF, Laravel menyarankan memakai Controller Tradisional.

Ketik perintah: php artisan make:controller PayrollPdfController

A. Logika Pembuat File PDF app/Http/Controllers/PayrollPdfController.php

<?php
namespace App\Http\Controllers;

use App\Models\Payroll;
use Barryvdh\DomPDF\Facade\Pdf;

class PayrollPdfController extends Controller
{
    public function download(Payroll $payroll)
    {
        // Panggil relasi nama orangnya
        $payroll->load('employee');

        // Render PDF dari Desain HTML bernama 'slip-gaji.blade.php' 
        $pdf = Pdf::loadView('pdf.slip-gaji', ['payroll' => $payroll]);

        // Tetapkan nama file yang didownload Otomatis!
        $filename = 'slip-gaji-' . $payroll->employee->name . '-' . $payroll->pay_date->format('F-Y') . '.pdf';

        return $pdf->download($filename);
    }
}
