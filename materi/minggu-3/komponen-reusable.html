3.3 Komponen Reusable

📕 Minggu 3 ⏱️ 20 menit

🎯 Tujuan Belajar

  • Membuat komponen yang bisa dipakai berulang
  • Passing data antar komponen
  • Best practices struktur kode

Contoh: Card Component

BookCard.php 
class BookCard extends Component
{
    public Book $book; // Type-hint model
    
    public function mount(Book $book)
    {
        $this->book = $book;
    }
}
book-card.blade.php 
<div class="card">
    <img src="{{ $book->cover }}">
    <h3>{{ $book->title }}</h3>
    <p>{{ $book->author }}</p>
</div>

Menggunakan Component

book-list.blade.php 
@foreach($books as $book)
    <livewire:book-card :book="$book" :key="$book->id" />
@endforeach
⚠️

Jangan lupa :key

Saat menggunakan komponen dalam loop, selalu tambahkan :key untuk performa optimal.

Best Practices

  • ✅ Satu komponen = satu tanggung jawab
  • ✅ Gunakan type-hint untuk property
  • ✅ Komponen kecil lebih baik dari komponen besar
  • ✅ Hindari nested component terlalu dalam
