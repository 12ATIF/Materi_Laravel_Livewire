3.1 Upload File Livewire

📕 Minggu 3 ⏱️ 25 menit

🎯 Tujuan Belajar

  • Upload gambar dengan Livewire
  • Preview gambar sebelum simpan
  • Validasi file (tipe, ukuran)

1. Setup Component

UploadCover.php 
use Livewire\WithFileUploads;

class UploadCover extends Component
{
    use WithFileUploads;
    
    public $cover;
    
    protected $rules = [
        'cover' => 'image|max:2048', // Max 2MB
    ];
    
    public function save()
    {
        $this->validate();
        
        $path = $this->cover->store('covers', 'public');
        
        // Simpan $path ke database
        session()->flash('success', 'Cover berhasil diupload!');
    }
}

2. Blade dengan Preview

upload-cover.blade.php 
<form wire:submit="save">
    <!-- Input file -->
    <input type="file" wire:model="cover">
    
    <!-- Loading state -->
    <div wire:loading wire:target="cover">
        Mengupload...
    </div>
    
    <!-- Preview gambar -->
    @if($cover)
        <img src="{{ $cover->temporaryUrl() }}" width="200">
    @endif
    
    <!-- Error message -->
    @error('cover') 
        <span class="error">{{ $message }}</span>
    @enderror
    
    <button type="submit">Simpan</button>
</form>
⚠️

Jangan lupa!

Jalankan php artisan storage:link supaya file bisa diakses dari public folder.

Validation Rules untuk File

Rule Fungsi
image Harus gambar (jpg, png, gif, dll)
max:2048 Max 2MB (dalam KB)
mimes:jpg,png Hanya jpg dan png
dimensions:min_width=100 Min lebar 100px
