3.1 Upload File dengan Livewire

📕 Minggu 3 ⏱️ 30 menit 🔧 Teori + Praktik

📌 Prasyarat

  • Sudah paham wire:model dan Validasi
  • Sudah bisa buat CRUD Sederhana

🎯 Tujuan Belajar

  • Upload gambar/file menggunakan WithFileUploads
  • Menampilkan preview sebelum disimpan
  • Validasi file (tipe, ukuran) & loading state
  • Menyimpan path file ke database

🤔 Kenapa Upload File Itu Spesial?

📦

Analogi: Kirim SMS vs Kirim Paket

  • 📱 Input teks (wire:model) = kirim SMS → kecil, langsung sampai
  • 📦 Upload file = kirim paket → perlu waktu, ada batas berat, perlu dicek
  • 🏪 storage/ = gudang penyimpanan paket
  • 🔗 storage:link = pintu masuk agar paket bisa diakses publik
1️⃣
User pilih file

Upload ke folder tmp

2️⃣
Preview muncul

temporaryUrl()

3️⃣
Simpan permanen

store() → storage/

🔨 Step 1: Class PHP

app/Livewire/UploadCover.php 
use Livewire\Component;
use Livewire\WithFileUploads;  // ← import trait
use App\Models\Book;

class UploadCover extends Component
{
    use WithFileUploads;  // ← aktifkan di dalam class

    public $cover;
    public $title = '';

    protected $rules = [
        'title' => 'required|min:3',
        'cover' => 'required|image|max:2048',  // Max 2MB
    ];

    protected $messages = [
        'cover.required' => 'Cover wajib diupload.',
        'cover.image'    => 'File harus gambar (JPG/PNG/GIF).',
        'cover.max'      => 'Ukuran maksimal 2MB.',
    ];

    public function save()
    {
        $this->validate();

        // ① Simpan file ke storage/app/public/covers
        $path = $this->cover->store('covers', 'public');

        // ② Simpan path ke database
        Book::create([
            'title' => $this->title,
            'cover_path' => $path,
        ]);

        $this->reset(['title', 'cover']);
        session()->flash('success', '✅ Buku berhasil disimpan!');
    }
}
🔍

Cara Kerja store()

  • 'covers' = subfolder (disimpan di storage/app/public/covers/)
  • 'public' = disk agar bisa diakses browser
  • Return: path seperti covers/abc123.jpg (nama otomatis random)

🔨 Step 2: View dengan Preview & Loading

resources/views/livewire/upload-cover.blade.php 
<div>
    @if(session()->has('success'))
        <div style="padding:1rem;background:#D1FAE5;color:#059669;border-radius:8px">
            {{ session('success') }}
        </div>
    @endif

    <form wire:submit="save">
        <div style="margin-bottom:1rem">
            <label>Judul Buku</label>
            <input wire:model="title">
            @error('title') <span style="color:red;font-size:0.85rem">{{ $message }}</span> @enderror
        </div>

        <div style="margin-bottom:1rem">
            <label>Cover</label>
            <input type="file" wire:model="cover" accept="image/*">

            <!-- Loading saat upload -->
            <div wire:loading wire:target="cover" style="color:#3B82F6;font-size:0.85rem">
                ⏳ Mengupload...
            </div>

            <img src="{{ $cover->temporaryUrl() }}"
                 style="width:200px;border-radius:8px;margin-bottom:1rem">
        @endif

        <button type="submit">
            <span wire:loading.remove wire:target="save">💾 Simpan</span>
            <span wire:loading wire:target="save">⏳ Menyimpan...</span>
        </button>
    </form>
</div>
⚠️

Wajib: Jalankan storage:link

Agar gambar bisa diakses browser: php artisan storage:link

Lalu tampilkan: <img src="{{ asset('storage/' . $book->cover_path) }}">

📚 Validation Rules untuk File

Rule Fungsi Contoh
image Harus gambar (jpg, png, gif, webp) 'cover' => 'image'
mimes:jpg,png Hanya ekstensi tertentu 'file' => 'mimes:pdf,doc'
max:2048 Max ukuran (KB): 2048=2MB max:5120 = 5MB
dimensions:min_width=100 Min lebar gambar 'foto' => 'dimensions:min_width=200'
file Harus berupa file 'doc' => 'file|max:10240'

📂 Upload File Non-Gambar

Upload PDF dengan nama custom 
public $dokumen;

protected $rules = [
    'dokumen' => 'required|file|mimes:pdf,doc,docx|max:5120',
];

public function upload()
{
    $this->validate();
    // storeAs = nama file custom
    $nama = 'tugas_' . time() . '.' . $this->dokumen->getClientOriginalExtension();
    $this->dokumen->storeAs('documents', $nama, 'public');
}
💡

store() vs storeAs()

store() = nama random otomatis (aman). storeAs() = nama custom yang kamu tentukan.

📎 Upload Multiple Files

Livewire juga mendukung upload banyak file sekaligus:

Multiple File Upload 
// Di class PHP
public $photos = [];  // Array, bukan single

protected $rules = [
    'photos'   => 'required',
    'photos.*' => 'image|max:2048',  // Validasi TIAP file
];

public function save()
{
    $this->validate();
    foreach ($this->photos as $photo) {
        $photo->store('gallery', 'public');
    }
}

// Di view (tambahkan "multiple")
<input type="file" wire:model="photos" multiple accept="image/*">

<!-- Preview semua file -->
@foreach($photos as $photo)
    <img src="{{ $photo->temporaryUrl() }}" style="width:100px">
@endforeach
💡

Perhatikan: photos.* (wildcard)

Rule 'photos.*' artinya "validasi setiap item di dalam array photos". Tanda * = wildcard untuk semua index.

📊 Upload Progress Indicator

Untuk file besar, tunjukkan progress bar agar user tahu proses upload-nya:

Progress bar menggunakan Alpine.js + Livewire events 
<div
    x-data="{ uploading: false, progress: 0 }"
    x-on:livewire-upload-start="uploading = true"
    x-on:livewire-upload-finish="uploading = false"
    x-on:livewire-upload-cancel="uploading = false"
    x-on:livewire-upload-error="uploading = false"
    x-on:livewire-upload-progress="progress = $event.detail.progress"
>
    <input type="file" wire:model="cover">

    <!-- Progress Bar -->
    <div x-show="uploading">
        <div style="background:#E5E7EB;border-radius:8px;overflow:hidden;height:8px">
            <div :style="'width:'+progress+'%;background:#3B82F6;height:100%'"></div>
        </div>
        <span x-text="progress+'%'"></span>
    </div>
</div>
📝

Catatan Guru RPL

Progress indicator ini menggunakan Alpine.js (sudah termasuk di Livewire v3). Event livewire-upload-progress otomatis dikirim oleh Livewire selama proses upload.

⚠️ Kesalahan Umum Pemula

❌ Lupa use WithFileUploads

Error: "Property must be of type string"

✅ Tambahkan use WithFileUploads; di dalam class.

❌ Lupa php artisan storage:link

File tersimpan tapi gambar tidak muncul di browser (icon rusak 🖼️❌).

✅ Jalankan perintah sekali di terminal.

❌ Pakai temporaryUrl() untuk PDF

temporaryUrl() hanya untuk gambar!

✅ Untuk non-gambar: {{ $dokumen->getClientOriginalName() }}

📋 Ringkasan

  • Trait WithFileUploads untuk mengaktifkan upload
  • $cover->temporaryUrl() untuk preview gambar
  • $cover->store('folder', 'public') untuk simpan
  • Jalankan php artisan storage:link
  • Validasi: image, mimes, max, dimensions
