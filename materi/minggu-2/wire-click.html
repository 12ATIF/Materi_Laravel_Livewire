2.2 Event (wire:click)

📘 Minggu 2 ⏱️ 20 menit

🎯 Tujuan Belajar

  • Handle event klik dengan wire:click
  • Passing parameter ke method
  • Mengenal event lain (submit, keydown, dll)

Apa itu wire:click?

wire:click memungkinkan kamu memanggil method PHP langsung dari tombol HTML. Tanpa perlu JavaScript!

Contoh Dasar

Counter.php 
class Counter extends Component
{
    public $count = 0;
    
    public function increment()
    {
        $this->count++;
    }
    
    public function decrement()
    {
        $this->count--;
    }
}
counter.blade.php 
<div>
    <button wire:click="decrement">-</button>
    <span>{{ $count }}</span>
    <button wire:click="increment">+</button>
</div>

Passing Parameter

Contoh dengan Parameter 
<!-- Blade -->
<button wire:click="setCategory('fiction')">Fiksi</button>
<button wire:click="setCategory('non-fiction')">Non-Fiksi</button>

// PHP
public function setCategory($category)
{
    $this->category = $category;
}

Event Lainnya

Event Kegunaan
wire:click Saat diklik
wire:submit Saat form disubmit
wire:keydown.enter Saat tekan Enter
wire:change Saat nilai berubah

📝 Latihan

  1. Buat component "Calculator" dengan tombol +, -, ×, ÷
  2. Tambahkan method reset() untuk mengembalikan nilai ke 0
