2.2 Event Handling (wire:click & lainnya)

📘 Minggu 2 ⏱️ 25 menit 📖 Teori + Praktik

📌 Prasyarat

  • Sudah paham wire:model (binding data input ke property)
  • Sudah paham Struktur Component (property, method, render)

🎯 Tujuan Belajar

Setelah menyelesaikan materi ini, kamu akan mampu:

  • Menggunakan wire:click untuk memanggil method PHP dari tombol HTML
  • Mengirim parameter ke method lewat wire:click
  • Menggunakan wire:submit untuk form
  • Mengenal event lain: wire:keydown, wire:change, dll
  • Menggunakan modifier event: .prevent, .self

🖱️ Apa itu wire:click?

Di materi sebelumnya, wire:model menghubungkan data (property). Sekarang wire:click menghubungkan aksi (method). Saat tombol diklik → method PHP dijalankan → tampilan diupdate. Tanpa JavaScript sama sekali!

🎮

Analogi: Remote TV

Bayangkan tombol remote TV:

  • Tombol Volume + → TV menaikkan volume → tampilan volume berubah
  • Tombol Channel ▲ → TV pindah channel → tampilan channel berubah

wire:click = tombol remote yang menjalankan "perintah" (method) di "TV" (server PHP)!

Alur wire:click

👆 User Klik
tombol
→
🔗 wire:click
kirim ke server
→
⚙️ Method PHP
proses logika
→
🔄 Re-render
update tampilan

📝 Contoh 1: Counter Sederhana

app/Livewire/Counter.php 
class Counter extends Component
{
    public $count = 0;

    public function increment()
    {
        $this->count++;
    }

    public function decrement()
    {
        if ($this->count > 0) {
            $this->count--;
        }
    }

    public function resetCounter()
    {
        $this->count = 0;
    }
}
resources/views/livewire/counter.blade.php 
<div style="text-align:center;">
    <h2>{{ $count }}</h2>

    <button wire:click="decrement">➖ Kurang</button>
    <button wire:click="resetCounter">🔄 Reset</button>
    <button wire:click="increment">➕ Tambah</button>
</div>

📦 Mengirim Parameter

Method bisa menerima parameter langsung dari view — sangat berguna untuk aksi pada item tertentu!

app/Livewire/BookFilter.php 
class BookFilter extends Component
{
    public $category = 'semua';

    // Method menerima parameter dari view
    public function setCategory($cat)
    {
        $this->category = $cat;
    }
}
resources/views/livewire/book-filter.blade.php 
<div>
    <h3>Filter Buku: <strong>{{ $category }}</strong></h3>

    <!-- Kirim string sebagai parameter -->
    <button wire:click="setCategory('semua')">📚 Semua</button>
    <button wire:click="setCategory('fiksi')">📖 Fiksi</button>
    <button wire:click="setCategory('sains')">🔬 Sains</button>
    <button wire:click="setCategory('sejarah')">🏛️ Sejarah</button>
</div>
🔍

Cara Kerja Parameter

  • wire:click="setCategory('fiksi')" → memanggil method setCategory dan mengirim string 'fiksi' sebagai argumen $cat
  • Bisa kirim angka juga: wire:click="setPage(2)"
  • Bisa kirim variabel: wire:click="delete({{ $book->id }})"

📋 wire:submit — Event Form

Untuk form, gunakan wire:submit alih-alih wire:click pada tombol submit:

Contoh wire:submit 
<!-- ✅ Cara yang Benar -->
<form wire:submit="simpan">
    <input wire:model="nama" placeholder="Nama">
    <input wire:model="email" placeholder="Email">
    <button type="submit">💾 Simpan</button>
</form>

<!-- ❌ Kurang tepat: pakai wire:click di tombol submit -->
<form>
    <input wire:model="nama">
    <button wire:click="simpan">Simpan</button>
</form>
💡

Kenapa wire:submit lebih baik?

  • wire:submit otomatis mencegah reload halaman (sama seperti e.preventDefault() di JavaScript)
  • User bisa submit form dengan menekan Enter, tidak hanya klik tombol
  • Livewire 3 otomatis menambahkan prevent ke wire:submit

🎛️ Event Lainnya

Selain click dan submit, Livewire mendukung banyak event HTML:

Event Livewire Trigger Contoh Penggunaan
wire:click Saat elemen diklik Tombol, link, card
wire:submit Saat form disubmit Simpan data, login
wire:keydown Saat tombol keyboard ditekan Shortcut keyboard
wire:keydown.enter Saat tekan Enter Submit chat, search
wire:keydown.escape Saat tekan Escape Tutup modal/popup
wire:change Saat nilai berubah Dropdown filter
wire:mouseenter Saat mouse masuk elemen Tooltip, preview

Contoh: Tekan Enter untuk Tambah Todo

Contoh wire:keydown.enter 
<input type="text"
       wire:model="newTask"
       wire:keydown.enter="addTask"
       placeholder="Tulis task lalu tekan Enter...">

🔀 Contoh Nyata: Tab Navigation

Gabungkan wire:click + parameter untuk membuat navigasi tab yang interaktif:

app/Livewire/ProfileTabs.php 
class ProfileTabs extends Component
{
    public $activeTab = 'profil';

    public function setTab($tab)
    {
        $this->activeTab = $tab;
    }
}
resources/views/livewire/profile-tabs.blade.php 
<div>
    <!-- Tab buttons -->
    <div style="display:flex;gap:0.5rem;">
        <button wire:click="setTab('profil')"
                class="{{ $activeTab === 'profil' ? 'active' : '' }}">
            👤 Profil
        </button>
        <button wire:click="setTab('nilai')"
                class="{{ $activeTab === 'nilai' ? 'active' : '' }}">
            📊 Nilai
        </button>
        <button wire:click="setTab('absensi')"
                class="{{ $activeTab === 'absensi' ? 'active' : '' }}">
            📅 Absensi
        </button>
    </div>

    <!-- Tab content -->
    @if($activeTab === 'profil')
        <div>Konten profil siswa...</div>
    @elseif($activeTab === 'nilai')
        <div>Konten nilai siswa...</div>
    @elseif($activeTab === 'absensi')
        <div>Konten absensi siswa...</div>
    @endif
</div>

⚡ wire:model vs wire:click — Kapan Pakai yang Mana?

Aspek wire:model wire:click
Fungsi Sinkronkan data Jalankan aksi
Trigger User mengetik / memilih User mengklik
Target Property Method
Contoh wire:model="nama" wire:click="simpan"
Elemen HTML input, select, textarea button, div, span, dll

⚠️ Kesalahan Umum Pemula

❌ Lupa tanda kutip pada parameter string

wire:click="setCategory(fiksi)" → ERROR! Livewire mengira fiksi adalah variabel.

✅ Solusi: Gunakan tanda kutip: wire:click="setCategory('fiksi')"

❌ Pakai wire:click di form padahal lebih tepat wire:submit

Tombol submit di dalam form sebaiknya menggunakan wire:submit di tag <form>.

✅ Solusi: Pindahkan handler ke form: <form wire:submit="simpan">

❌ Method tidak public

Method private atau protected tidak bisa dipanggil dari view!

✅ Solusi: Pastikan semua method yang dipanggil dari Blade adalah public function

❌ Nama method "reset" konflik dengan Livewire

reset() adalah method bawaan Livewire. Kalau kamu buat public function reset(), bisa bentrok!

✅ Solusi: Pakai nama lain seperti resetCounter(), resetForm()

📋 Ringkasan Materi

  • wire:click = memanggil method PHP saat elemen diklik
  • Bisa mengirim parameter: wire:click="method('value')"
  • wire:submit = handler form yang lebih tepat dari wire:click di tombol submit
  • Event lain: wire:keydown.enter, wire:change, wire:mouseenter
  • wire:model = untuk data (property), wire:click = untuk aksi (method)
