2.3 Validasi Livewire

📘 Minggu 2 ⏱️ 25 menit

🎯 Tujuan Belajar

  • Membuat validasi form dengan Livewire
  • Menampilkan pesan error real-time
  • Menggunakan rules validation Laravel

Validasi Dasar

ContactForm.php 
class ContactForm extends Component
{
    public $name = '';
    public $email = '';
    public $message = '';
    
    protected $rules = [
        'name' => 'required|min:3',
        'email' => 'required|email',
        'message' => 'required|min:10',
    ];
    
    public function submit()
    {
        $this->validate();
        
        // Proses data...
        session()->flash('success', 'Pesan terkirim!');
    }
}

Validasi Real-Time

Dengan method updated(), error muncul saat user mengetik:

ContactForm.php 
public function updated($propertyName)
{
    $this->validateOnly($propertyName);
}

Menampilkan Error di Blade

contact-form.blade.php 
<form wire:submit="submit">
    <div>
        <input wire:model.live="name" placeholder="Nama">
        @error('name') 
            <span class="error">{{ $message }}</span>
        @enderror
    </div>
    
    <div>
        <input wire:model.live="email" placeholder="Email">
        @error('email') 
            <span class="error">{{ $message }}</span>
        @enderror
    </div>
    
    <button type="submit">Kirim</button>
</form>

Rules Populer

Rule Fungsi
required Wajib diisi
email Format email valid
min:3 Minimal 3 karakter
max:255 Maksimal 255 karakter
unique:users,email Email unik di tabel users
