2.1 Binding Data (wire:model)

📘 Minggu 2 ⏱️ 25 menit 📖 Teori + Praktik

📌 Prasyarat

  • Sudah paham Struktur Component Livewire (property, method, render)
  • Sudah paham Blade Template ({{ }}, @if, @foreach)
  • Mengerti HTML form dasar: input, textarea, select, checkbox

🎯 Tujuan Belajar

Setelah menyelesaikan materi ini, kamu akan mampu:

  • Menjelaskan konsep data binding dan alur datanya
  • Menggunakan wire:model untuk menyinkronkan input form dengan property PHP
  • Memilih modifier yang tepat: .live, .blur, .debounce
  • Menggunakan wire:model dengan berbagai jenis input (text, select, checkbox, dll)

🤔 Apa itu Data Binding?

Di materi sebelumnya, kamu sudah belajar bahwa property di class PHP bisa ditampilkan di view pakai {{ $nama }}. Tapi bagaimana kalau user ingin mengubah nilai property lewat input form?

Di sinilah wire:model berperan — dia menjadi "jembatan" yang menyinkronkan nilai input di browser dengan property di server.

📱

Analogi: Google Docs

Bayangkan kamu sedang mengedit dokumen di Google Docs:

  • Kamu ketik di browser → teks otomatis tersimpan di cloud (server Google)
  • Teman yang membuka dokumen yang sama → langsung melihat perubahanmu

wire:model bekerja mirip: ketika user mengetik di input → nilai di server PHP otomatis terupdate → tampilan di browser ikut berubah!

Alur Data wire:model

⌨️ User Ketik
di <input>
→
🔗 wire:model
kirim ke server
→
🖥️ Property PHP
$nama = "Budi"
→
🔄 Re-render
update tampilan

📝 Contoh Dasar: Salam Otomatis

Kita buat komponen sederhana — user ketik nama, langsung muncul salam!

app/Livewire/Greeting.php 
namespace App\Livewire;

use Livewire\Component;

class Greeting extends Component
{
    // Property yang akan di-bind ke input
    public $nama = '';

    public function render()
    {
        return view('livewire.greeting');
    }
}
resources/views/livewire/greeting.blade.php 
<div>
    <label>Masukkan namamu:</label>
    <input type="text"
           wire:model.live="nama"
           placeholder="Ketik nama...">

    @if($nama)
        <p style="font-size:1.5rem;margin-top:1rem;">
            👋 Halo, <strong>{{ $nama }}</strong>! Selamat belajar!
        </p>
    @endif
</div>
🔍

Penjelasan

  • wire:model.live="nama" — menghubungkan input ini ke property $nama di PHP. Setiap ketikan langsung dikirim ke server
  • @if($nama) — hanya tampilkan salam kalau user sudah mengetik sesuatu
  • {{ $nama }} — tampilkan nilai terbaru dari property

✨ Hasilnya: Ketik "Budi" → langsung muncul "Halo, Budi! Selamat belajar!" tanpa reload!

⚙️ Modifier wire:model

Secara default, wire:model di Livewire 3 hanya mengirim data saat form disubmit (deferred). Kamu bisa mengubah perilakunya dengan modifier:

Modifier Kapan Data Dikirim Contoh Penggunaan
wire:model Saat form disubmit (default Livewire 3) Form registrasi, form kontak
wire:model.live Real-time setiap ketikan Search bar, preview langsung
wire:model.blur Saat input kehilangan fokus (klik di tempat lain) Validasi email saat pindah field
wire:model.live.debounce.500ms Real-time tapi delay 500ms Search yang hemat request
wire:model.live.debounce.1s Real-time tapi delay 1 detik Auto-save draft
🎯

Kapan Pakai Modifier Mana?

  • Form biasa (registrasi, kontak) → wire:model saja. Hemat request!
  • Search bar → wire:model.live.debounce.300ms. Real-time tapi tidak terlalu sering kirim.
  • Preview live → wire:model.live. Langsung terlihat saat mengetik.
  • Validasi per field → wire:model.blur. Cek saat user pindah ke field berikutnya.

Contoh: Search dengan Debounce

app/Livewire/BookSearch.php 
class BookSearch extends Component
{
    public $search = '';

    public function render()
    {
        $books = [];

        if (strlen($this->search) >= 2) {
            $books = Book::where('title', 'like', '%' . $this->search . '%')->get();
        }

        return view('livewire.book-search', [
            'books' => $books
        ]);
    }
}
resources/views/livewire/book-search.blade.php 
<div>
    <input type="text"
           wire:model.live.debounce.300ms="search"
           placeholder="🔍 Cari judul buku...">

    @forelse($books as $book)
        <div class="book-item">
            <strong>{{ $book->title }}</strong>
            <span>oleh {{ $book->author }}</span>
        </div>
    @empty
        @if(strlen($search) >= 2)
            <p>😕 Buku tidak ditemukan.</p>
        @endif
    @endforelse
</div>

🔤 wire:model dengan Berbagai Jenis Input

wire:model tidak hanya untuk <input type="text">! Berikut contoh lengkapnya:

Form lengkap — berbagai jenis input 
<!-- Text input -->
<input type="text" wire:model="nama">

<!-- Email input -->
<input type="email" wire:model="email">

<!-- Number input -->
<input type="number" wire:model="umur">

<!-- Textarea -->
<textarea wire:model="pesan"></textarea>

<!-- Select / Dropdown -->
<select wire:model="kelas">
    <option value="">-- Pilih Kelas --</option>
    <option value="X">Kelas X</option>
    <option value="XI">Kelas XI</option>
    <option value="XII">Kelas XII</option>
</select>

<!-- Checkbox -->
<input type="checkbox" wire:model="setuju"> Saya setuju

<!-- Radio buttons -->
<input type="radio" wire:model="gender" value="L"> Laki-laki
<input type="radio" wire:model="gender" value="P"> Perempuan
🔍

Tipe Data Property

Jenis Input Tipe Property PHP Nilai Default
text, email, textarea string '' (string kosong)
number int atau float 0
checkbox bool false
select, radio string ''

⚠️ Kesalahan Umum Pemula

❌ Mengira wire:model langsung real-time (Livewire 3)

Di Livewire 3, wire:model default-nya adalah deferred (data dikirim saat submit). Di Livewire 2, default-nya real-time.

✅ Solusi: Tambahkan .live kalau mau real-time: wire:model.live="search"

❌ Nama property tidak cocok dengan wire:model

wire:model="nama" harus persis sesuai dengan public $nama di class. Typo = error!

✅ Solusi: Pastikan nama di wire:model 100% sama dengan nama property di PHP. Case-sensitive!

❌ Pakai .live di semua input form

Menambahkan .live di setiap input = banyak request ke server. Bikin lambat!

✅ Solusi: Gunakan .live hanya untuk search atau preview. Untuk form biasa, wire:model tanpa modifier sudah cukup.

📋 Ringkasan Materi

  • wire:model = menyinkronkan input HTML dengan property PHP di server
  • Default Livewire 3: data dikirim saat submit (deferred)
  • .live = real-time, .blur = saat pindah fokus, .debounce = delay
  • Bisa dipakai untuk semua jenis input: text, email, textarea, select, checkbox, radio
  • Nama di wire:model harus persis sama dengan nama property di class PHP

  3. Tambahkan @if($setuju) untuk menampilkan tombol submit hanya kalau checkbox dicentang.
