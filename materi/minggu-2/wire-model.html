2.1 Binding Data (wire:model)

📘 Minggu 2 ⏱️ 25 menit

🎯 Tujuan Belajar

  • Memahami konsep data binding
  • Menggunakan wire:model untuk input form
  • Mengenal modifier .live, .blur, .debounce

Apa itu wire:model?

wire:model adalah cara Livewire untuk menyinkronkan nilai input dengan property PHP. Ketika user mengetik, nilai langsung terupdate di server!

💡

Analoginya

Seperti chat WhatsApp yang langsung terkirim saat kamu mengetik, tanpa perlu tekan tombol "Kirim".

Contoh Dasar

Greeting.php 
class Greeting extends Component
{
    public $nama = '';
}
greeting.blade.php 
<div>
    <input type="text" wire:model="nama" placeholder="Ketik nama...">
    <p>Halo, {{ $nama }}!</p>
</div>

Modifier wire:model

Modifier Penjelasan
wire:model Default, update saat blur (pindah fokus)
wire:model.live Update real-time saat mengetik
wire:model.blur Update saat input kehilangan fokus
wire:model.live.debounce.500ms Update real-time tapi delay 500ms
Contoh Modifier 
<!-- Real-time update -->
<input wire:model.live="search">

<!-- Dengan debounce 300ms -->
<input wire:model.live.debounce.300ms="search">
📌

Kapan pakai .live?

Gunakan .live untuk fitur search real-time. Untuk form biasa, cukup wire:model saja supaya hemat request.

📝 Latihan

  1. Buat component "Profil" dengan input nama dan email
  2. Tampilkan preview "Nama: [nama], Email: [email]"
  3. Coba bedakan wire:model biasa dengan wire:model.live
