2.4 CRUD Sederhana

📘 Minggu 2 ⏱️ 35 menit 🔧 Praktik

🎯 Tujuan Belajar

  • Memahami konsep CRUD (Create, Read, Update, Delete)
  • Membuat to-do list sederhana dengan Livewire

Apa itu CRUD?

➕
Create

Tambah data

👁️
Read

Baca data

✏️
Update

Ubah data

🗑️
Delete

Hapus data

Contoh: To-Do List

TodoList.php 
class TodoList extends Component
{
    public $todos = [];
    public $newTodo = '';
    
    // CREATE
    public function addTodo()
    {
        $this->todos[] = [
            'text' => $this->newTodo,
            'done' => false
        ];
        $this->newTodo = '';
    }
    
    // UPDATE
    public function toggleTodo($index)
    {
        $this->todos[$index]['done'] = !$this->todos[$index]['done'];
    }
    
    // DELETE
    public function removeTodo($index)
    {
        unset($this->todos[$index]);
        $this->todos = array_values($this->todos);
    }
}
todo-list.blade.php 
<div>
    <!-- Form tambah -->
    <form wire:submit="addTodo">
        <input wire:model="newTodo" placeholder="Tugas baru...">
        <button>Tambah</button>
    </form>
    
    <!-- List todos -->
    @foreach($todos as $index => $todo)
        <div>
            <input type="checkbox" 
                   wire:click="toggleTodo({{ $index }})"
                   {{ $todo['done'] ? 'checked' : '' }}>
            <span>{{ $todo['text'] }}</span>
            <button wire:click="removeTodo({{ $index }})">🗑️</button>
        </div>
    @endforeach
</div>
📌

Catatan

Contoh di atas menyimpan data di property (hilang saat refresh). Di materi selanjutnya, kita akan simpan ke database!
