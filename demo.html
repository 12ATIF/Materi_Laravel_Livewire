class Counter extends Component
{
    public int $count = 0;
    
    public function increment()
    {
        $this->count++;
    }
    
    public function decrement()
    {
        $this->count--;
    }
}

üìù Form Validation - wire:model

Validasi real-time saat user mengetik. Error muncul langsung!

RegisterForm.php 
class RegisterForm extends Component
{
    public $name = '';
    public $email = '';
    public $password = '';
    
    protected $rules = [
        'name' => 'required|min:3',
        'email' => 'required|email',
        'password' => 'required|min:6',
    ];
    
    public function updated($field)
    {
        $this->validateOnly($field);
    }
}
